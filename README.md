# Simple project to run jasmine tests

<p>karma-cli is required</p>

<h2>Installation</h2>
<div class="highlight highlight-source-shell">
<pre>
    git clone https://github.com/xcintas/github.git
    cd github
    npm install
</pre>
</div>

<h2>Usage</h2>
<div class="highlight highlight-source-shell">
<pre>
    karma start
</pre>
</div>
